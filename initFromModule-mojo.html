opencms:initFromModule

Full name:

org.synyx:maven-opencms:1.2-SNAPSHOT:initFromModule

Description:

Creates the module structure for an existing module. Copies all included files, creates properties as well as acl entries according to the manifest.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
pathToModule String - The absolute path to the module zip to import from.

Optional Parameters

Name Type Since Description
targetPath String - The path to create the folder structure to.
Default value is: src/main/vfs.

Parameter Details

pathToModule:

The absolute path to the module zip to import from.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${opencms.pathToModule}

targetPath:

The path to create the folder structure to.
  • Type: java.lang.String
  • Required: No
  • Expression: ${opencms.targetPath}
  • Default: src/main/vfs
